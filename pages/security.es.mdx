# El camino hacia la producción

<Callout type="warning" emoji="🏗">
  Madara está todavía en fase *experimental*. Así que ten cuidado a la hora de pasar a producción con un sistema 
  de este tipo, se recomienda encarecidamente realizar auditorías exhaustivas.
</Callout>

Madara pretende ser seguro *por defecto* y luego dejar la responsabilidad a los ingenieros de ejecución para asegurar 
que los cambios que hacen mantienen el sistema seguro.
Esta es la razón principal por la que Madara se basa en software seguro y listo para la producción, como Substrate, 
blockifier y cairo-rs. Ya asegura miles de millones de dólares de valor en diferentes cadenas.
